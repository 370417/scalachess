Chess API written in scala with minimal dependencies

INSTALL
-------

1. Clone scalachess

    git clone git://github.com/ornicar/scalachess

2. Get latest sbt (0.11.3) 

http://www.scala-sbt.org/download.html

3. In scalachess, run

    sbt

You're in. In the sbt shell, to compile, run

    compile

To run the tests:

    test
